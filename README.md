# K2-Plus
## Improvements and setups
## Mejoras y configuraciones


- **Extruder LED.**  
Turn on and off the led in the extruder.  
Enciende y apaga el LED en el extrusor.  


- **Horizon levelling.**  
Simplify manual bed tramming, ensuring the bed is perfectly parallel to the X and Y axes.
Hace fácilmente el ajuste clásico de las tuercas por debajo de la cama, para que la cama esté paralela a los ejes x e y.  

  
- **0º bed plate.**  
Print with the bed heater disabled, using a cold plate.  
Posibilita imprimir con la cama apagada utilizando una bandeja fría.  

  
- **Skew correction.**  
Obtain your skew and save here.  
Obten tu skew y grábalo aquí.  

  
- **Z Ofsset.**  
Don't lost it, save as variable.  
Que no se borre, guárdalo como una variable.  

  
- **Fast levelling.**  
Fast levelling in every print.  
Nivelado rápido en cada impresión.  

  

More explanations in each folder.  
Más explicaciones en cada carpeta.
